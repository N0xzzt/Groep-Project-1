# Groep-Project-1
Wij zijn Team Ctrl+Alt+Delete, een verzameling half-functionerende ICT-gremlins die samen zijn gegooid om een project te bouwen voordat onze hersencellen rage-quitten. We communiceren vooral in code, koffiemorsen en ongepaste “werkt niet?? ja fix het dan”-kreten. Iedereen heeft een rol, maar niemand weet eigenlijk wat die rol doet. Onze Scrum-master is eigenlijk gewoon een menselijke Google Calendar, de programmeur spreekt uitsluitend in stackoverflow-dialect, en de designer heeft 47 tabbladen open waarvan er maar één relevant is.

Onze missie?
Een opdracht bouwen die nét goed genoeg werkt zodat de docent zegt: “Ja… dit is technisch gezien functioneel.”
Ons motto?
If it compiles, we vibe. If it crashes, we blame the JSON.

Samen vormen wij een chaotische maar charmante glitch in de Matrix. En hé, zolang Git niet ontploft, komt het allemaal wel goed.
Misschien.
